名词解释

交叉编译：
　　指在A平台(如windows)上编译生成，在B平台(如arm)上运行的过程。这个两个平台具有不同的机器指令。

　　如果不采用交叉编译，我们在A平台（如windows）上编译出来的可执行文件只能运行在X86的CPU下，无法在B平台（如arm）上运行。
工具链：由编译器、连接器、解释器和调试器组成 　(没有编辑器)
